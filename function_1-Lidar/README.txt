Dossier regroupant les codes liés à l'application du lidar :
- documentation : informations générales sur le lidar
- ros programs (Jetson) : ensemble du code implémenté dans la jetson
- ros programs (Raspberry) : ensemble du code implémenté dans la raspberry


Lancement de l'application Lidar :
- sur la Jetson : lancer launch_lidar.sh
- sur la Raspberry : lancer rosrun recup_data_lidar Ecriture.py
